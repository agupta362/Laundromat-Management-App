# Laundromat Management App

A console-based Python application designed to manage laundry transactions for a small laundromat.

## Features
- Calculate laundry costs based on weight
- Apply tax rates to transactions
- Track cumulative customer spending
- View customer transaction history
- Input validation for user input

## Technologies Used
- Python
- Dictionaries and functions
- Console-based interface

## Usage
Run the program using:
```bash
python laundromat.py
